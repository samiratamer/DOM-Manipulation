# DOM Manipulation – Article Filter & Add

A JavaScript web application that dynamically filters and adds articles using DOM manipulation, without any page reloads.

## Features

**Filter Articles**
- Clicking the "Filter Articles" button reveals a dropdown menu with category checkboxes
- Checking or unchecking a category instantly shows or hides all articles of that type

**Add New Article**
- Clicking the "Add New Article" button displays a form for article entry
- Submitting the form dynamically adds the new article to the list with correct styles applied

## Concepts Used
- DOM selection and manipulation
- Event listeners and click handling
- Dynamic element creation and insertion
- Conditional show/hide logic
- Form input handling
- CSS class-based styling applied via JavaScript

## How to Run
Open `index.html` in any web browser — no installation or setup required.
